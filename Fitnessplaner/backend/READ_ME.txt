1. Vergewissern Sie sich, dass Sie Node.js installiert haben. 

Die offizielle Webseite -> https://nodejs.org/en 

Für das Projekt wurde die Version 18.15.0 LTS verwendet. 

2.  Ich empfehle Ihnen “Visual Studio Code” zu verwenden.  

Wenn das Projekt in Visual Studio Code geöffnet ist, sollte das Terminal geöffnet werden, sollte im backend-Ordner folgende Eingabe erfolgen: 

->	cd backend 

-> 	npm init -y  

->	npm i express				 

-> 	npm i body-parser 

->	npm i concurrenty 

->	npm i ejs 

->	npm i express-ejs-layouts 

->	npm i mongoose@5.13.16 

->	npm i multer 

-> 	npm i nodemon
